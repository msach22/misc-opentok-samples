Session Swap Test

In this example, we test moving a publisher from one session to another without removing it from the DOM.

Note: first copy config.sample.js to config.js and input your API and generated sessions/tokens

  1. Connect to either of the sessions below. The publisher will publish to whichever session is connected.
  2. Open this URL in another window (or from another machine) and connect to the other session. Nothing has changed yet.
  3. Change the session from either of the windows. Now both are connected to the same session and should be subscribing to each other.
